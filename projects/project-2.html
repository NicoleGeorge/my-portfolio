Workday Planner

The brief was to create a simple calendar application that allows the user to save events for each hour of the day. The app runs in the browser and features dynamically updated HTML and CSS powered by jQuery. I used the Moment.js library to work with date and time.

Our objective: An employee, with a busy schedule, needs to be able to add important events to a daily workday planner, enabling them to manage their time effectively. Local storage was used to save user data.
