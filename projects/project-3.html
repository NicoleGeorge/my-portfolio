travel blogger:
weather dashboard

A travel blogger wants to see the weather outlook for multiple cities to plan their next trip. They also want to know the 5-day forecast of the cities they are looking to visit.

The objective was to build a weather dashboard, using the Open Weather API to pull weather data from searched cites. The output runs in the browser, and features dynamically updated HTML and CSS, powered by JQuery.

App Functionality:

current forecast

User searches for a city by entering a city name into the search field. The search is validated if no city is entered but a search is attempted, the user will be prompted to 'please enter a city name to continue'. The user is presented with the city's current weather conditions, for various cities. These cities will output dynamically under 'My wanderlust'.

5-day forecast

The user clicks on the '5-Day forecast' link in the header and the 5-day forecast will be displayed. The min & max temperatures for each city searched. As above, cities search by the user will be added to their 'My wanderlust' notepad. these can also be removed.
